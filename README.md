# BackEnd Polo-IT

## Description

Este es el backend un sistema de gestión de inscripciones a proyectos, desarrollado con Nodejs, TypeScript, Express y Mysql

## Tecnologías

- Nodejs
- Javascript/Typescript  
- Express
- PostgreSQL
- Prisma ORM
- Zod
- Swagger
- Biome
- Jest & SuperTest
- Docker
- Jenkins
